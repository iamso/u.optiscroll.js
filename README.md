u.optiscroll.js
====

u.js plugin for Optiscroll.

Install
-------

### With Bower

```bash
bower install u.optiscroll.js
```

### With Components

```bash
component install u.optiscroll.js
```

Usage
-----
Check out [Optiscroll](https://github.com/wilsonfletcher/Optiscroll).
